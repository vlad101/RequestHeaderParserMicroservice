API Basejump: Request Header Parser Microservice

Instructions:
  • The API returns the IP address, and operating system (language, software) for your browser.

Example usage:

http://{{main.host}}:{{main.port}}/api/whoami

Example output:

{ "ipaddress": "24.103.245.182", "language": "en-US", "software": "Windows NT 6.3; WOW64" }
